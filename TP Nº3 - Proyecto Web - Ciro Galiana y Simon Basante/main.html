Como jugar:

Objetivo: Ganar dos de las tres rondas o “bazas” que componen la mano.

Reparto: Se reparten tres cartas a cada jugador.

Mecánica:

  • Cada jugador tira una carta por ronda. La carta más alta gana la baza.
  • El equipo o jugador que gana la primera ronda comienza tirando en la segunda.
  • El que gana dos de las tres bazas gana la mano (el “truco”).
  • Apuestas del Truco: Se puede “cantar” Truco en cualquier momento antes de que finalice la mano.
    • Truco: Vale 2 puntos. Se puede responder con:
      • “No quiero”: El que cantó Truco gana 1 punto.
      • “Quiero”: Se juega por 2 puntos.
    • Retruco: Sube la apuesta a 3 puntos. Se puede responder con:
      • “No quiero”: El que cantó Retruco gana 2 puntos.
      • “Quiero”: Se juega por 3 puntos.
    • Vale Cuatro: Sube la apuesta a 4 puntos. Se puede responder con:
      • “No quiero”: El que cantó Vale Cuatro gana 3 puntos.
      • “Quiero”: Se juega por 4 puntos.
